# This Folder is supposed to contain single feature
```
.
└── src/
    └── features/
        └──auth
            └──Login.tsx
            └──Register.tsx
            └──api.ts
```

Each feature should be in its own folder named after the feature. The folder should contain all related components, utilities, and API calls (e.g., `Login.tsx`, `Register.tsx`, `api.ts`).
