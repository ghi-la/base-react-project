# This Folder is supposed to contain single components
```
.
└── src/
    └── components/
        ├── navbar/
        │   ├── Navbar.tsx
        │   └── Navbar.css
        └── card/
            ├── Card.tsx
            └── Card.css
```

Each component should be in its own folder named after the component. The folder should contain the component file (e.g., `Navbar.tsx`) and any related styles or assets (e.g., `Navbar.css`).
